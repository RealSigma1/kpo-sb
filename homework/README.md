# Moscow Zoo Management System

Консольное приложение для автоматизации учета животных и инвентаря Московского зоопарка.

## Функционал

### Основные возможности
- Добавление новых животных с проверкой здоровья
- Ветеринарный контроль перед приемкой
- Расчет общего количества потребляемой еды
- Формирование списка животных для контактного зоопарка
- Учет животных и вещей с инвентарными номерами
- Раздельные отчеты по животным и инвентарю

### Дополнительные возможности
- Изменение норм питания для животных
- Добавление различных типов инвентаря
- Подробные отчеты с фильтрацией
- SOLID архитектура с DI контейнером

## Архитектура решения

### SOLID принципы
- **SRP**: Каждый класс имеет одну ответственность
- **OCP**: Легко добавлять новые виды животных и вещей
- **LSP**: Наследники корректно заменяют родительские классы  
- **ISP**: Интерфейсы разделены (IAlive, IInventory)
- **DIP**: Dependency Injection через ServiceContainer

### Система животных
- **Травоядные**: Monkey, Rabbit (доброта > 5 для контактного зоопарка)

- **Хищники**: Tiger, Wolf (не могут в контактный зоопарк)
  Автоматическая проверка здоровья перед добавлением
